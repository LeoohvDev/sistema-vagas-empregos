# 💼 Sistema de Vagas de Emprego

Sistema de gerenciamento de vagas de emprego desenvolvido em JavaScript puro, utilizando `prompt` e `alert` para interação com o usuário.

---

## 🚀 Como rodar o projeto

1. Abra o arquivo `sistemaVagas.html` no navegador.

2. O sistema será iniciado automaticamente via `prompt`.

---

## ⚙️ Funcionalidades

- ✅ Listar vagas disponíveis
- ✅ Criar uma nova vaga
- ✅ Visualizar detalhes de uma vaga
- ✅ Inscrever um candidato em uma vaga
- ✅ Excluir uma vaga

---

## 🛠️ Tecnologias usadas

- JavaScript puro (Vanilla JS)
- HTML

---

## 📚 O que aprendi

- Manipulação de arrays de objetos
- Métodos de array: `.map()`, `.filter()`, `.find()`, `.push()`, `.splice()`, `.join()`
- Estruturas de repetição: `do while` com `break` e `continue`
- Funções nomeadas e anônimas
- Escopo de variáveis
- Conversão de tipos com `parseInt()`
